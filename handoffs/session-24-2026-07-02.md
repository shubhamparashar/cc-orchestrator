## v1.14.1 — verified bug sweep (PRs #49, #50)

- **Server:** JSON `null`/scalar bodies → 400; client aborts treated as disconnects (not 500s into `lastError`//api/diag); `/api/send` cwd type-validated; SSE broadcast caps per-client buffering at 4 MB and destroys dead clients; `safeHandler` guarantees every request path answers (pre-route auth throws used to hang requests); cost-rollup + reindex failures logged instead of swallowed.
- **Live sessions (`lib/liveSessions.mjs`):** stop escalates SIGTERM→SIGKILL after grace; `prune()` never evicts non-exited entries (no unkillable orphans); spawn failure fires onExit exactly once; **`AnsiStreamStripper`** (`lib/ansi.mjs`) + streaming TextDecoder carry partial escape sequences/code points across PTY chunk boundaries.
- **Data:** `digest()` skips `<synthetic>` records (context gauge/model no longer zeroed/mislabeled); `dateUsageCache` pruned; ctx-generate survives stdin EPIPE and always releases the gen-lock.
- **Dashboard (12 guards):** Send prompt keeps the typed draft + shows the error on non-2xx (was silently discarded); stale-response guards on chat viewer / launcher / history palette / new-live context picker; board freshness counter (slow poll can't clobber SSE data); repo selects don't rebuild while focused; Enter-on-match respects IS_REMOTE; truthful clipboard feedback; drafts don't leak across sessions.
- **Security (review nits fixed pre-merge):** `/login` failure bucket separated from generic unauth traffic — a stale dashboard tab can no longer lock out a **correct-token** login (brute-force limits intact, e2e-tested); proxy identity headers (`Tailscale-User-Login`/XFF) trusted **only from a loopback socket** (LAN header-spoof bucket rotation closed); 500-logs record path without query (no `/login?key=…` credential in the log).
- Tests **167 → 187** (auth-ratelimit e2e, csrf, liveSessions/AnsiStreamStripper, scan-digest, cost-prune).

## v1.15.0 — modern UI design system (PRs #51, #52)

Ground-up restyle of the single-file frontend (~940 lines changed), still zero-dep/no-build:

- **Tokens:** flat elevation `--elev-1/2/3`, hairline `--line` borders, spacing + type scales, `color-mix` badge/chip system — clay shadows and ~40 hardcoded colors gone.
- **Dark theme default** (OS-following, leaning dark) + sun/moon toggle, `localStorage('cc-theme')`, pre-paint boot script, synced theme-color metas, theme-native live terminal with streaming caret.
- **Cards:** 3px status rail, primary badge row + `+N` expander, `⋯` action menu (IS_REMOTE gating preserved), softer hover; **keyed per-card DOM reconcile** (DJB2 hash over `cardHtml`, keyed `data-sid`) keeps j/k selection, expansion, and open menus stable across SSE refreshes.
- **Header:** scroll-aware elevation (sentinel + IntersectionObserver), segmented filters, mobile `⋯` tools panel; sticky colheads pin below a ResizeObserver-measured `--header-h`.
- **States/motion:** skeletons, designed empty states with clear-filters, dialog enter/exit transitions + pinned dlghead/scrollable dlgpad, all gated by a global `prefers-reduced-motion` kill-switch.
- **A11y/mobile:** WCAG AA in both themes (light `--muted`/`--blue` darkened post-review to clear 4.5:1 on chips), `:focus-visible`, ≥44px targets, 16px inputs (no iOS zoom), safe-area header, sheet grab-handle.
- Also fixed pre-existing breakage: ⌘K chip CSS collision, undefined `var(--border)`, board empty-state in column 1, off-palette hexes.
